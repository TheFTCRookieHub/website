Don't Do This!

Hardware Horrors

  • Using Standard Nuts: Robots VIBRATE. Standard nuts fall off in 10 seconds.
    Fix: ONLY use Nyloc Nuts (the ones with the white plastic insert inside).
  • Tiny Screws (Stripping): Using the wrong size hex key ruins screws forever.
    Fix: Buy high-quality Wera or Bondhus hex keys.
  • No Strain Relief: Cables plugged directly into the Hub get yanked out when the robot moves.
    Fix: 3D print cable retaining clips or simple zip-ties to secure wires to the chassis.

Software Slip-ups

  • The "Big Bang" Code: Writing 500 lines of code before testing if the wheels turn.
    Fix: Test small. Make one motor spin. Then two. Then drive. Then intake.
  • "The robot isn't done yet": Waiting for the mech team to finish perfectly before coding.
    Fix: Program on a spare chassis or just a hub with motors on a table.
