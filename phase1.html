The First 30 Days

Kickoff & Strategy (Week 1)

Coach Pratt's Tip: "Do not build a single thing until you have read the manual and decided on a strategy."

Read Game Manual Part 2: This tells you how to score. Memorize it.

The Strategy Flow:

  1. Analyze: What actions earn points? (e.g., parking, placing blocks).
  2. Prioritize: Rank tasks by [Points per Second]. Don't chase hard tasks for low points.
  3. Design: Sketch mechanism ideas (Claw? Intake rollers? Lift?).

Choosing Your Hardware

Most teams today rely on a "Build System". Mixing them is hard for rookies.

REV Robotics

Uses 15mm extrusion. Very adjustable (slide anywhere). Comes with the "Control Hub".

goBILDA

Uses "Pattern" based channel. Extremely sturdy and rookie-friendly because holes always line up.

Programming: Where to Start?

You have three main choices. Pick one and stick to it.

  • Blocks: Like Scratch. Drag and drop.
    Best for absolute true beginners.
  • OnBot Java: Text-based Java in the browser.
    Best for teams who want to learn Java without installing software.
  • Android Studio: Professional IDE.
    Powerful but high learning curve. Recommended for Year 2+.
